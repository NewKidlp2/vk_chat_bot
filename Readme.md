# Запуск приложения:

## JDK:
1. Установите jdk (минимум 8 версия) с [сайта](https://www.oracle.com/java/technologies/javase-downloads.html)
2. Откройте переменные среды: откройте панель управления -> система и безопасность -> система -> дополнительные
параметры системы -> переменные среды.
3. Убедитесь, что директория bin JDK добавилась в переменную PATH (системная переменная, для этого необходимо
открыть переменные среды и открыть переменную PATH, если её нет, то необходимо добавить.
## ngrok:
1. Скачайте ngrok с [сайта](https://ngrok.com/download) и запустите.
2. В открывшейся командной строке наберите `ngrok http {*port_number*}`, где `{port_number}`, это свободный порт
на Вашем устройстве (по умолчанию 8081).
## Файл конфигураций:
1. Откройте файл конфигураций `application.properties`, находящийся по пути `{*project folder*}/src/main/resources`.
2. В качестве порта `server.port` укажите порт, указанный вами при настройке ngrok.
3. В поле `server.url` введите `url`, сгенерированный программой `ngrok`.
4. Не вносите изменений в названия переменных и в остальные поля.
## Непосредственно запуск:
1. Откройте командную строку и переместитесь в директорию проекта.
2. Введите команду `gradle run`. Для выполенения этой команды необходим установленный сборщик Gradle.
